このサイトの使い方

① 基本の使い方

・基礎ATK / DEF を入力
・リーダースキルを入力
・パッシブスキルを入力
・必殺技倍率を設定
→ 「計算」を押す

② パッシブスキルの書き方

以下の形式で入力してください： 
基本効果
・ATK/DEF150%UP

攻撃時
・ATK100%UP

数値を含まない行は見出しとして扱われます
例：
基本効果
攻撃時

先頭に「-」をつけると無効化されます

例：
-基本効果 → その見出しを無効
-・ATK150%UP → その行のみ無効

ATK/DEFを含まない数値は計算に影響しません
例：
HP50%以下 → 無視されます

③ パッシブ判定ルール

● パッシブ1（常時系）
基本効果 / 登場時 / ターン開始など

● パッシブ2（条件系）
攻撃時 / 攻撃を受けるとき / 回避するたび など

④ 耐久ラインについて

完封ライン：ダメージ0になる敵ATK
耐久ライン：HP内で耐えられる最大敵ATK

⑤ 注意事項

・実際のゲームとは完全一致ではありません
・あくまで理論値・目安です
